# Sai Driving School & Travels

A responsive and user-friendly website for a driving school and travel service platform. This project provides information about services like driving training, car rentals, and car sales, along with features for user inquiries and bookings.

## Features

- Modern and responsive UI  
- Driving course details (beginner to advanced)  
- Car rental and travel services  
- Car buying and selling section  
- Contact form for inquiries  
- Service pricing display  
- Clean navigation and layout
